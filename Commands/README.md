
# Nmap Command Reference

This directory contains categorized Nmap commands used throughout this project.

The commands are grouped by their purpose, making it easier to understand when and why each scanning technique should be used during an authorized security assessment.

## Contents

- Host Discovery
- Port Scanning
- Service Enumeration
- Operating System Detection
- NSE (Nmap Scripting Engine)
- Firewall Evasion Techniques
- Timing Templates
- Output Formats

## Purpose

The objective of this reference is to provide a practical command library for learning Nmap and documenting common techniques used in penetration testing, vulnerability assessments, and network administration.

> **Disclaimer**
>
> All examples are intended for use only in authorized environments where you have explicit permission to perform security testing.
